# Real time strategie Game

- there are many players(1v1)
- each one can only maipulate ones owne troups
- the troups can move or not
- there are (many) kinds of troups

## Requirements
Minimum requirements:
- you can join a server (1v1)
- one person can win the game
- you can leave the game
- you can interact with entitys:
    - entitys are "owned" by a player
    - there are static and dynamic entitys
    - dynamic (aka moving) entitys can be orderd
    - static yield resources and defence
    - one static entity might be the "king piece" (aka the win condition)

Median requirements:
- make menu
    - main menu
    - joining menu
- auto finding game on LAN

Maximum requirements:
- traking gaming stats (win count)

Impossible requirements:
- make it look nice
- make the graphics good
